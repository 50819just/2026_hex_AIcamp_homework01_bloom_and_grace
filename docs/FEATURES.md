# FEATURES.md

## 目前功能

### 1. 作業首頁
- 顯示第一場作業的整理版畫面
- 提供綠界金流串接說明
- 顯示測試流程與注意事項

### 2. 綠界測試訂單建立
- 輸入姓名、Email、商品名稱、金額
- 建立本地測試訂單
- 導向綠界測試付款頁

### 3. 付款結果查詢
- 付款完成後可回到本地前端頁面
- 以前端觸發 server 查單
- 顯示訂單狀態、付款方式、付款時間與原始回應

### 4. AI Agent 協作文件
- `AGENTS.md`
- `docs/DEVELOPMENT.md`
- `docs/ARCHITECTURE.md`
- `docs/FEATURES.md`
- `docs/TESTING.md`
- `docs/plans/archive/*`
