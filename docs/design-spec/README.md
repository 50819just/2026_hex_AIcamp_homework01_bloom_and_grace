# Design Spec 放置區

這個資料夾用來放與產品設計、使用者流程、驗收導向需求有關的規格文件。

## 目前文件
- `DESIGN_SPEC.md`：本次六角作業版本的主要設計規格，包含購物流程、登入限制、綠界付款、Admin 驗收區與測試文件需求。

## 使用方式
- 想確認「這個專案現在真正要做什麼」時，先看 `DESIGN_SPEC.md`。
- 想確認技術規範與開發邊界時，再對照：
  - `docs/DEVELOPMENT.md`
  - `docs/ARCHITECTURE.md`
  - `docs/FEATURES.md`
  - `docs/TESTING.md`
