# TESTING.md

## 基本驗證

### 1. 前端建置
```bash
npm run build
```

### 2. 啟動本地服務
```bash
npm run server
npm run dev
```

### 3. 綠界付款測試流程
1. 開啟前端首頁
2. 填入測試訂單資料
3. 點擊「建立測試訂單並前往付款」
4. 進入綠界測試付款頁
5. 使用綠界測試信用卡付款
6. 付款後返回本地結果頁
7. 點擊或自動觸發查單，確認付款成功

## 測試卡資訊
- 信用卡號：`4311-9522-2222-2222`
- 安全碼：任意三碼
- 有效年月：大於現在月份
- 3D 驗證簡訊：`1234`

資料來源：綠界測試介接資訊官方文件。

## 注意事項
- 本地端通常收不到 `ReturnURL` 的 server 通知。
- 本作業以 `OrderResultURL` 返回前端後，再由本地 server 呼叫 `QueryTradeInfo` 查單確認。
- 若查不到資料，先確認：
  - `MerchantTradeNo` 是否正確
  - 主機時間是否正確
  - 測試環境金鑰與特店編號是否一致
